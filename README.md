# CustomSelectItem

第一步 在项目里的build.gradle里加上 
<br>
maven { url 'https://jitpack.io' }
<br>
<br>
第二部 在app的build.gradle里加上 
<br>
compile 'com.github.lyman86:CustomSelectItem:v1.0.1'  


若是嫌点击范围不够大，可以动态设置   
<br>
lyman:clickRangeEx="20dp"
